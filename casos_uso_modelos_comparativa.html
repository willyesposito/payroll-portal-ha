⚙️ Cómo varía pedirle el mismo artifact a cada modelo
El tipo de artifact lo elegís vos según el resultado que querés. El modelo (Haiku, Sonnet, Opus) afecta sobre todo la calidad del diseño, profundidad del código y velocidad. Misma pregunta, distinto pulido:
Haiku 4.5
Rápido pero artifacts más simples y planos. Ideal para iterar borradores o cuando la velocidad importa más que el pulido.
Sonnet 4.6
Balance ideal — diseño profesional, código limpio y tiempos razonables. La opción default para casi todo.
Opus 4.7
Mejor calidad visual, anticipa problemas de UX y maneja artifacts complejos con muchos componentes. Para entregables finales.
